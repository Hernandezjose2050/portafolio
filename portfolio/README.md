# Portafolio Carlos Hernandez

Portafolio personal profesional construido con HTML5, CSS3 y JavaScript Vanilla.

## Estructura

```text
portfolio/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
├── images/
│   ├── avatar.svg
│   ├── project-api.svg
│   ├── project-currency.svg
│   ├── project-ecommerce.svg
│   └── project-inventory.svg
└── README.md
```

## Funcionalidades

- Diseño oscuro responsive con glassmorphism.
- Navbar fija con menú hamburguesa en móvil.
- Scroll suave y enlaces activos por sección.
- Efecto typing en el hero.
- Animaciones al hacer scroll.
- Barras de progreso animadas.
- Validación de formulario con JavaScript.
- Botón para volver arriba.
- Fondo con partículas usando canvas y JavaScript puro.

## Uso

Abre `index.html` directamente en el navegador o usa la extensión Live Server de VS Code.

## Personalización

- Cambia enlaces reales de GitHub, LinkedIn y correo en `index.html`.
- Reemplaza `images/avatar.svg` por una foto o avatar propio.
- Actualiza los enlaces de demo y repositorio en las tarjetas de proyectos.
- Agrega tu CV real en `images/cv-carlos-hernandez.pdf` si quieres activar la descarga.
